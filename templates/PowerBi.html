About PowerBi Project

The Benefits of Data Analyzing and Data Visualization are becoming increasingly evident to business organisation around the world. Power BI is business intelegence service that is employed by many of these organisations to implement these usefull tchniques on there business data maintained in the cloud data warehouse and numerous other sourses. User use Power BI reports to visualiza there data and perform analysis after introducing power BI and its various Formates we can understand from the details coming from Power BI reports after embedment we can publish the reports. So thats why Power BI is coleective group of an assortment of services and cloud based apps that can help organisation at all level to manage , analyze the data from different Data Sources easily with the help of very user freindly interface. The Data sources can be various data bases and Microsoft Excel as well. After Preprocessing It helps in making very usefull reports by providing easy to use visualization and very interactive interface. Here In this Project we can Analyze , visualize the data of car through "Microsoft Power BI"and Predict the price of car by using "Machine Learning Model".
Power BI exist in the following Formates :

  • Windows desktop is known as Power BI Desktop
